Discovery Package Management System

DPMS is a modern, lightweight package manager built specifically for Discovery Linux. It makes finding, installing, and updating software incredibly fast, reliable, and straightforward.

How it Works

🎯

Safe & Isolated

DPMS is designed to keep your system stable. Every installation is transactional and isolated, ensuring that updating one application won't accidentally break another.

⚡

Built for Speed

By using parallel downloads and smart registry caching, DPMS syncs with repositories and resolves complex dependencies up to three times faster than older package managers.

Core Features

Custom Repositories

Easily expand your software sources by adding custom, isolated third-party repositories without messing up core system files.

GPG Security Checks

Rest easy knowing every single package is automatically scanned and verified using GPG cryptographic signatures before installation.

Smart Dependency Resolution

No more dependency hell. DPMS intelligently maps out exactly what an application needs and safely installs the correct versions.

Installation & Setup

curl -fsSL https://dpms.sh | sudo bash
git clone https://github.com/Discovery-linux/dpms.git && cd dpms && make native-release
sudo apt install dpms -y
