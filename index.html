What is DPMS?

🎯 Core Philosophy

DPMS (Discovery Package Management System) is built on the principle that package management should be: fast, modular, and accessible. It replaces traditional package managers with a modern, architecture-aware system that understands your environment.

⚡ Why DPMS?

  • ✓ 3x Faster package discovery than legacy managers
  • ✓ Intelligent dependency resolution engine
  • ✓ Three interfaces: CLI, TUI, GUI
  • ✓ Minimal footprint on system resources
  • ✓ Modular architecture for extensibility

Three Interface Modes

⌨️

CLI Mode

Lightning-fast command-line interface for power users and automation. Perfect for scripts, CI/CD pipelines, and terminal professionals.

dpms install --recursive python3
📟

TUI Mode

Interactive terminal user interface with intuitive menus and real-time feedback. Great for system administrators managing packages visually.

Navigate • Search • Install • Manage
🖥️

GUI Mode

Beautiful graphical interface for desktop users. Point-and-click package management with visual package information and system monitoring.

Beginner-friendly • Visual • Modern

System Mechanics

🔍

Smart Discovery

Automatically discovers your system architecture and dependencies. Intelligent package resolution engine that understands complex dependency chains.

📦

Modular Architecture

Extensible plugin system for custom package sources. Create custom package repositories and repositories with built-in distribution tools.

⚡

Lightning Fast

Optimized for speed with parallel downloads and caching. Minimal resource footprint - runs efficiently even on constrained systems.

🔐

Secure & Verified

GPG signature verification for all packages. Cryptographic integrity checks and secure package sources out of the box.

🔄

Smart Updates

Intelligent update management with rollback capabilities. Only update what changed - minimize system disruption.

📊

Real-time Monitoring

Live package dependency graphs and system health monitoring. Track installation progress and system impact in real-time.

Installation Methods

Quick Install
curl -fsSL https://dpms.sh | sudo bash
Build from Source
git clone https://github.com/Discovery-linux/dpms.git cd dpms && make install
Via Package Manager
sudo apt install dpms

✓ System Requirements: Linux kernel 5.4+, glibc 2.31+, 512MB disk space

✓ Supported Architectures: x86_64, ARM64, ARMv7, RISC-V
